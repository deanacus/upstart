The Stack component is a layour component that allows you to stack things in a 
vertical or horizontal manner, with an even amount of space between them.

```jsx
  <Stack>
    <DemoBox>Item</DemoBox>
    <DemoBox>Item</DemoBox>
    <DemoBox>Item</DemoBox>
    <DemoBox>Item</DemoBox>
  </Stack>
```
