There are 3 components that make up Lists in Upstart:

<UnorderedList>
  <ListItem>UnorderedList</ListItem>
  <ListItem>OrderedList</ListItem>
  <ListItem>ListItem</ListItem>
</UnorderedList>

## Unordered List

The `UnorderedList` provides a simple interface to an unordered list, as well as
an option to use a custom list icon, and indentation width.

```jsx
<UnorderedList>
  <ListItem>List item 1</ListItem>
  <ListItem>List item 2</ListItem>
  <ListItem>List item 3</ListItem>
  <ListItem>List item 4</ListItem>
</UnorderedList>
```

### Props

<PropsTable component={UnorderedList} />

## OrderedList

As with the `UnorderedList`, the `OrderedList` component provides an interface
to an ordered list, a way to define the ordering scheme, and indentation width.

```jsx
<OrderedList>
  <ListItem>List item 1</ListItem>
  <ListItem>List item 2</ListItem>
  <ListItem>List item 3</ListItem>
  <ListItem>List item 4</ListItem>
</OrderedList>
```

### Props

<PropsTable component={OrderedList} />

## ListItem

The fundamental buuilding block of a list is, of course, the `ListItem`.
Accepting no props at this stage, this component is simply a wrapper.

```jsx
<ListItem>Google</ListItem>
<ListItem>Facebook</ListItem>
<ListItem>Twitter</ListItem>
```
