As well as simple links, we have created a component that will return a range of
differently styled and behaving link elements.

```jsx
<p>Lorem ipsum dolor <Link href="/components/link">sit amet</Link>, consectetur
adipisicing elit.</p>
```

<PropsTable component={Link} />

## External

```jsx
<p>Lorem ipsum dolor <Link href="https://google.com" isExternal>sit amet</Link>, consectetur
adipisicing elit.</p>
```
