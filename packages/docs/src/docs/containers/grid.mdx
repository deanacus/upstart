The Upstart Grid is a na√Øve implementation of CSS grid, that allows an arbitrary
number of grid columns, each `1fr` in width. There is no grid-row handling, due
to the added complexity that would add, plus the fact that if you need a specific
number of rows and columns, you are probably better off defining your own grid.

## Default Grid

By default, the Grid has 4 columns, and a grid-gap of 1.6rem. Here it is:
